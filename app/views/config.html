Environment

Enter to the Environment field the base URL of your PureCloud region, e.g. mypurecloud.ie for Europe.

Client ID

Follow below administrative steps to configure a new oAuth application:

  • Login to the PureCloud using administrative credentials.
  • Navigate to: Admin -> Integrations -> OAuth.
  • Click Add Client button.
  • Enter following settings:
    • App Name: Colima
    • Grant Types: Token Implicit Grant (Browser)
    • Authorized redirect URIs: http://localhost:9000/
      (or other valid URI in case if youâ€™re hosting the application on different server)
  • Click Save button.
  • Copy a new generated Client ID and paste it into the Client ID field on this page.
